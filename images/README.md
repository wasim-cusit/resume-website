# Images Directory

This directory contains images used in the resume website.

## Required Images:

1. **profile.jpg** - Profile picture for the About section
   - Recommended size: 300x300 pixels
   - Format: JPG or PNG
   - Should be a professional headshot

## How to add images:

1. Place your profile image in this directory
2. Name it `profile.jpg` (or update the HTML to match your filename)
3. Ensure the image is optimized for web (compressed, appropriate size)

## Note:

If you don't have a profile image, you can:
- Use a placeholder service like https://via.placeholder.com/300x300
- Create a simple avatar using online tools
- Remove the image element from the HTML temporarily
